# WhatsApp Contact Form Auto Template Sender

This project automatically sends a WhatsApp template message (with image + variable) when a user submits a contact form.

## Setup

1. Install dependencies:
```
npm install
```

2. Create `.env` file:
```
WHATSAPP_TOKEN=YOUR_PERMANENT_TOKEN
PHONE_NUMBER_ID=YOUR_PHONE_NUMBER_ID
TEMPLATE_NAME=contact_acknowledgement
```

3. Start server:
```
npm start
```

Visit:
http://localhost:3000/contact.html
